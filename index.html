About

Forensics Without
Machine Learning

DevLens uses deterministic, interpretable rule-based logic to detect synthetic images. Unlike neural network approaches, every detection decision can be traced back to specific measurable image properties.

The system evaluates four independent forensic signals: pixel intensity statistics, blur/smoothness profiles, edge gradient consistency, and facial symmetry ratios — each weighted and combined into a final confidence score.
